¿Qué es el web scraping?

El web scraping es una técnica utilizada para extraer datos de sitios web, haciendo uso de la estructura de HTML de una pagina web, esto con el fin de encontrar elementos específicos que sean de interés para la aplicación que lo implementa. Se podría asimilar como un robot recolector de información que permite obtener datos de la web sin necesidad de hacerlo manualmente.
