Tipos de aplicaciones web

Tipo Descripción
Aplicaciones web estaticas Las aplicaciones web estáticas son sitios diseñados para presentar información sin permitir interacción más allá de la lectura. Son simples, generalmente compuestos de HTML y CSS, y se usan comúnmente para sitios personales de profesionales. Aunque pueden incluir imágenes o videos, se mantienen simples para diferenciarse de otros tipos de aplicaciones web.
Aplicaciones web dinamicas Son sitios en la red que no se mantienen estáticos a lo largo del tiempo y que están en una proceso constante de acutalización que hace que haya diferentes contenidos cada que se visitan.
Para poder actualizarse constantemente requieren una mayor programación tanto con HTML y CSS con PHP o JavaScript y MySQL.
Aplicaciones web de comercio Son una de las más populares en el mercado digital de hoy. Esto se deben a que cada vez son más las empresas y personas que prefieren el uso de plataformas digitalespara comercializar sus mercancías. Un buen sitio web de comercio electronico debe ser al mismo tiempo una aplicación web dinamica.
Aplicaciones web de gestion de contenido Los CMS son un tipo de app web accesible desde navegador. Sirve para administrar los contenidos que muestran en un sitio web así como en cada una de sus páginas.
Aplicaciones web de pagina unica Este tipo de pagina opera completamente dentro de una sola pagina que solamente necesita actualizarse para mostrar cambios. Todas las acciones suceden dentro de la misma ventana.
Aplicaciones web de portal Este tipo de aplicacion contiene varios enlaces a diferentes sitios, funcionando como portal a varias paginas. Se pueden encontrar en paginas de empresas grandes que pueden ofrecer varios servicios por lo que tiende a ser util tener un portal que ayude a los usuarios encontrar una pagina especifica.
Aplicaciones web progresivas Estas estan hechas con un formato y funcionalidad similar a su contraparte en dispositivos moviles
